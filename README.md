# Flask Portfolio & Comments Scratchpad

A full-stack Flask web app deployed on PythonAnywhere.

## Features
- Portfolio profile page with CV, skills, experience and contact form
- Login-protected comments scratchpad backed by SQLite
- User authentication with hashed passwords
- Flask-Login, Flask-SQLAlchemy, Flask-Migrate

## Stack
Python · Flask · SQLite · SQLAlchemy · PythonAnywhere
